## 프로젝트 구성 

  ### 기획 및 프로젝트 기간
  
  - 기획기간 : 2023.03.03 ~ 03.05
  - 개발기간 : 2022.03.06 ~ 2022.03.24
  
  ### 프로젝트 목표
  - 서울 각 구의 행복지수, 문화환경 만족도, 대중교통 만족도, 병원 수 등 데이터 기반, 사용자가 원하는 살기좋은 지역을 찾아주는 서비스 제공
  - 주제 : 서울 지역의 살기 좋은 곳을 찾아주는 사이트
